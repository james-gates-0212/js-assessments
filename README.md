## 🐞Git commit

Please use [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) for your commits.

e.g.

```commit
<type>[scope]: <subject>
[body]
[footer]
```

### ♻️Types

`type` should be one of belows.

```bash
add
adopt
apply
build
chore
config
docs
feat
fix
init
refactor
remove
style
```
